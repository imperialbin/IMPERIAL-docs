# Create document

Theres a couple of ways in which you can create a document within our Visual Studio Code extension.

Through default keybinds:

Windows: `CTRL + Shift + S`

Linux: `CTRL + Shift + S`

macOS: `CMD + Shift + S`

Or, when selecting text, press the IMPERIAL icon at the top right

Or, when selecting text, right click, and press `Upload selected code`
