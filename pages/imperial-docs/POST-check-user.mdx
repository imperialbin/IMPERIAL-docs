# POST /checkUser

Check if a user is valid and get their name and profile picture

## The request

import Callout from 'nextra-theme-docs/callout'

#### Something to keep in mind

<Callout>You only get 30 requests every 20 seconds.</Callout>

#### The body

```json
username: "devlooskie" // Must be a string and is a requirement whilst requesting
```

#### Headers

```json
authorization: 'IMPERIAL-xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx'. // Must be a string. (Default: null)
```

## Response

These are the following responses you will receive.

#### Status code 200

Your request was successful

```json
success: true,
username: "devlooskie",
userPfp: "URL to PFP"
```

#### Status code 400

When you don't provide valid username in the request

```json
success: false,
message: "You must pass in username!"
```

#### Status code 404

When user doesn't exist

```json
success: false,
message: "That user does not exist"
```

#### Status code 429

You are being ratelimited, chill out for around 20 seconds.

#### Status code 500

There was an internal server error getting the user

```json
success: false,
message: "An error occurred whilst getting user"
```
