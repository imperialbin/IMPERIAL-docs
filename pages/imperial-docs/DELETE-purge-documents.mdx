# DELETE /purgeDocuments

Purge all documents by IMPERIAL's API

## The request

import Callout from 'nextra-theme-docs/callout'

#### Something to keep in mind

<Callout>You only get 30 requests every 20 seconds.</Callout>
<Callout>If your request isn't authorized, it WILL get REJECTED.</Callout>
<Callout>Once you send this request and it succeeds, we WILL NOT be able to get back your documents!</Callout>

#### Headers

```json
authorization: 'IMPERIAL-xxxxxxxx-xxxx-4xxx-yxxx-xxxxxxxxxxxx'. // Must be a string. (Default: null)
```

## Response

These are the following responses you will receive.

#### Status code 200

Deleted the document successfully

```json
success: true,
message: "Deleted a total of x documents!"
numberDeleted: x // number
```

#### Status code 400

When you don't have any documents to delete

```json
success: false,
message: "There was no documents to delete!"
```

#### Status code 401

When you aren't authorized

```json
success: false,
message: "Please put in a valid API token!"
```

#### Status code 429

You are being ratelimited, chill out for around 20 seconds.

#### Status code 500

There was an internal server error getting your user

```json
success: false,
message: "An internal server occurred whilst getting user"
```


There was an internal server error getting all your documents

```json
success: false,
message: "An internal server occurred whilst getting documents!"
```
