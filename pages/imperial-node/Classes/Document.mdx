# The Document Class

Gets returned from all of the [`Imperial`](./Imperial) methods that involve Documents from the Api.

## Constructor

You shouldn't explicitly use this to create a Document. It won't interact with the Api for you.

## Properties / Getters

#### `.content`

Returns: `String` - Content of the Document

Alias: `.code`

---

### `.formattedLink`

Returns: `String` - URL of the Document, to view in Imperial

---

### `.rawLink`

Returns: `String` - URL to a plain text version of the Document

---

### `.id`

Returns: `String` - Id of the Document

Alias: `.documentId`

---

### `.instantDelete`

Returns: `Boolean` - Whether the Document will be deleted after being viewed

---

### `.encrypted`

Returns: `Boolean` - Whether the document is encrypted

---

### `.views`

Returns: `Number` - Current view count of the Document

---

### `.editors`

Returns: `Array<String>` - List of allowed editors of the Document

Alias: `.allowedEditors`

---

### `.imageEmbed`

Returns: `Boolean` - Whether the Document will embed an image of the content

---

### `.langauge`

Returns: `String` or `null` - The Programming langauge that was set to the Document

---

### `.longerUrls`

Returns: `Boolean` - Wheather is the Document URL longer

---

### `.password`

Returns: `String` or `null` - Password for the Document

---

### `.creation`

Returns: `Date` - The date that the Document was created at

Alias: `.creationDate`

---

### `.expiration`

Returns: `Date` - The date that the Document will be deleted at

Alias: `.expirationDate`

---

### `.daysLeft`

Returns: `Number` or `null` - The ammount of days the Doucment will expire at from the current moment

---

### `.raw`

Returns: [`RawDocument`](../Types/RawDocument) - The raw JSON data of the Document

## Methods

#### `.delete()`

Deletes the current Document!

Token **is** required to be set in the [`Imperial`](./Imperial#constructor) Class.

Method takes no parameters.

Returns: `Promise<void>`

Alias: `.explode`

---

#### `.duplicate(options?)`

Duplicates the current Document!

<Callout emoji="ðŸ’¡">
  By default it copies all of the current Documents settings, but they can be
  changed with the `options` parameter
</Callout>

Token **is** required to be set in the [`Imperial`](./Imperial#constructor) Class.

| Parameter  |                     Type                      |
| :--------: | :-------------------------------------------: |
| `options?` | [`DocumentOptions`](../Types/DocumentOptions) |

Returns: `Promise<Document>`

---

### `.edit(text)`

Edits the content of the current Document!

Token **is** required to be set in the [`Imperial`](./Imperial#constructor) Class.

| Parameter |   Type   |
| :-------: | :------: |
|  `text`   | `String` |

Returns: `Promise<void>`
