# The Imperial Class

The easiest way to interact with the Api in node!

## Constructor

The constructor takes a `token` it's optional for basic methods but required for most. <br/>
The package will try to get the value of `IMPERAIL_TOKEN` from the `process.env`.

```ts
import { Imperial } from 'imperial-node'

const client = new Imperial()
```

## Properties / Getters

#### `.token`

Returns: `String` - Token, that was set in the constructor

---

#### `.hostname`

Returns: `String` - Imperial's hostname

---

#### `.hostnameCheckRegExp`

Returns: `RegExp` - Regular Expression that is used to match against in functions

## Methods

#### `.createDocument(text, options?)`

Creates a Document!

Token **is not** required.

| Parameter  |                     Type                      |
| :--------: | :-------------------------------------------: |
|   `text`   |                   `String`                    |
| `options?` | [`DocumentOptions`](../Types/DocumentOptions) |

Returns: `Promise<Document>`

Note, that the Document reffers to the [`Document`](./Document) class.

---

#### `.getDocument(id, password?)`

Gets a Document from Imperial!

Token **is not** required.

|  Parameter  |       Type        |
| :---------: | :---------------: |
|    `id`     | `String` or `URL` |
| `password?` |     `String`      |

Returns: `Promise<Document>`

Note, that the Document reffers to the [`Document`](./Document) class.

---

#### `.deleteDocument(id)`

Deletes a Document from Imperial!

Token **is** required.

| Parameter |       Type        |
| :-------: | :---------------: |
|   `id`    | `String` or `URL` |

Returns: `Promise<void>`

---

#### `.editDocument(id, text)`

Edits the Documents content!

Token **is** required.

| Parameter |       Type        |
| :-------: | :---------------: |
|   `id`    | `String` or `URL` |
|  `text`   |     `String`      |

Returns: `Promise<Document>`

Note, that the Document reffers to the [`Document`](./Document) class.

---

#### `.verify`

Verify if your token is valid!

Token **is** required, obviously.

Method takes no parameters.

Returns: `Promise<void>`

---

#### `.purgeDocuments`

Delete **all** of your created Documents.

Token **is** required.

Method takes no parameters.

Returns: `Promise<void>`
